7-zip: Free compression tool for Windows.

Git: Free, state of the art distributed version control system.

Visual Studio Express 2012 for Windows Desktop: Free, state of the art IDE and compilers for Windows development. Includes support for native C++11 applications and .NET desktop applications using C# and VB.NET (supports WPF).

Visual Studio Express 2012 for Windows 8: Free, state of the art IDE and compilers for development of Windows Store applications. Includes support for native C++11, Direct3D and .NET applications for the Windows Store (supports the Windows 8 native XAML library).

Visual Studio Express 2012 products home: Includes links for downloading all the Visual Studio Express tools.

Paint.NET: Free image and photo editing software for computers that run Windows.
